字符串的一些常用方法

in 、not in判断字符串中是否包含指定的字符

.capitalize() 首字母转为大写其他英文字母转为小写

.swapcase() 大写转为小写,小写转为大写

.lower() 全部转为小写

.upper() 全部转为大写

.title() 每个单词的首字母转为大写

.count() 统计字符串中字符出现的次数

.endswith() 判断字符串是否以某字符结尾

.startswith() 判断字符串是否以某字符开头

.find() 查找字符串的位置

.center() 给字符串前后添加空格或字符并让给定字符串居中

.replace() 替换给定字符

.strip() 删除前后空格或移除前后给定字符
