Python的基本数据类型———List

发表于 2019-02-18
字数统计: 1.5k | 阅读时长 ≈ 6

列表(list)是python中重要的数据类型，有点类似其它语言的数组，python中的列表可以是任意对象的集合

定义一个列表

可以在[ ]中使用逗号隔开对象来定义列表

在python中列表很灵活，具有以下特点

  • 可以包含任意对象
  • 可以无限嵌套
  • 列表是有序的
    试想一下[1,2,3]和[3,2,1]相等吗
  • 可以通过索引和切片的方式访问列表中的元素
    列表和string一样可以通过索引和切片的方式里面的元素，在列表中也可以使用负数的方式访问
