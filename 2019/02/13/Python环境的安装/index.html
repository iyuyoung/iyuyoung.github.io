发表于 2019-02-13

本系列是我会重新学习python，这次学习会带着大量的问题去学，在这里会记录我遇到的问题，以及我的解决方法。

什么是Python

Python是一种面向对象的解释型编程语言，由荷兰人Guido van Rossum于1989年圣诞节假期编写

为什么要学Python

由于人工智能正在以不可阻挡的趋势发展，Python更是挤进编程语言Top5。如果你喜欢简单、优雅、想用更少的代码做更多的事，那就不妨来试试Python,优秀的Python代码，读起来像是在读一篇优美的文章。

下面我们就来开启Python的大门，领略Python之美

到这里Python在几个平台上的安装就分享完了，但是你可能会觉得在一个小黑框里写Python代码很不方便，所以再和大家分享两款工具：Anaconda和PyCharm

Anaconda

Anaconda 是一种Python免费增值开源发行版，用于进行大规模数据处理、 预测分析、科学计算, 致力于简化包的管理和部署。Anaconda使用软件包管理系统Conda进行包管理,下载后直接双击安装。使用起来也是很方便，大家有兴趣可以去安装以后自己玩玩

PyCharm

PyCharm是由大名鼎鼎的捷克公司JetBrains开发，可能是最好用的Python开发IDE，但是这个软件比较贵，一些土豪可以直接花钱购买，如果和我一样穷可以选择社区版，但是机智的你总能想到办法。

总结：本篇分享了Python、pip以及python虚拟环境的安装,安装步骤都比较简单，需要重点了解一下python虚拟环境，你可以反复多启动几个虚拟环境练练手，如果你使用了Ubuntu16.04及以上,Python3和Python2都是自带的，不需要安装，开箱即用
